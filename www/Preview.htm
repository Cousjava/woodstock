Project Woodstock - Preview
Home Page > View and try interactive examples of the Woodstock components

Use the links below to preview the different Woodstock components.
 
Accordion (New!) The Accordion component is much like a sideways Tab, with vertical sliding behavior for opening the currently selected section while closing the previously open section.
ActionMenu (New!) The ActionMenu component is a menu that is attached to some component, like a Table or an Accordion element, and will "pop" open when triggered, usually by an icon.
Add-and-Remove Idiom The Add-and-Remove component, which lets users move list items between an Available items list and a Selected items list, and reorder the selected items.
Alert The Alert Message component that notify users of a condition or error.
BubbleHelp (New!) The BubbleHelp component is a bubble-like popup window that appears when triggered by a mouse-hover event for selected page element. It provides a bubble to place any applications specified content.
Button The Button component, including the ability to dynamically enable and disable buttons.
Calendar The Calendar component provides a full calendar widget for selecting dates from a popup month view calendar display
Checkbox and Radio Button The Checkbox (multiple or independent selection) and Radio Button (select one from group) components.
Common Tasks Page The Common Tasks Page component, which lets users choose among and launch the most commonly used tasks for an application.
Content Page Title The Content Page Title component, plus page buttons, page actions, and page view controls.
Dropdown Menu and Scrolling List The Drop-down Menu and Scrolling List components, including the ability to dynamically enable and disable them with client-side JavaScript technology.
Editable List The Editable List component, which allows users to add to and maintain a custom list of items.
File Chooser and Uploader The File/Folder Chooser and File Uploader components, which lets users select files and folders on the server, or upload a file on the client.
Hyperlink The Hyperlink component, which let users take links to move to related or detail locations.
Image and Masthead The Image and Masthead components, which show images and build the entire masthead -- like the one shown at the top of this page, but with the option to also add buttons and status information.
Label The Label component, including appropriate context-specific styles to use for a hierarchy of appearances with levels of visual weight.
Orderable List The Orderable List component, which enables users to rearrange (reorder) a list of elements.
ProgressBar The Progress Bar components, which let users see the proportion of progress toward a long-running task, or just see that the system is busy.
Property Sheet The Property Sheet component, which is a collection of controls and information about a particular object.
Static Text The Static Text component (text on the page), including date format masks and a custom converter.
Tab Set The Tab set component, which lets users choose among related subsections of information.
Table The Table component, including the ability to page or scroll tables of data, sort table data, select items, perform actions, and view alarms.
Text Input The Text field, Text area (a multiple line field), and Password components, including the ability to dynamically enable and disable them with client-side JavaScript technology.
Tree The dynamic and client-side Tree components, which allow users to browse through and choose from a hierarchy of information, including event handling and tree-based navigation.
Wizard The pop-up Wizards, which help the user proceed through a multiple step complex action via a pop-up window with previous and next buttons.
